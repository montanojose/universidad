# Proyectos universitarios

Este repositorio reúne proyectos desarrollados durante mi formación universitaria en programación, desarrollo web, bases de datos e ingeniería de software.

Los proyectos permiten observar mi evolución desde mi primera página web hasta el desarrollo de sistemas más completos y la aplicación de tecnologías como blockchain.

## Proyectos

### EduData

Primer proyecto de desarrollo web realizado durante el primer año de la carrera para la materia Programación Web.

Aunque se trata de un proyecto básico y presenta aspectos que actualmente resolvería de otra manera, representó mi primera experiencia construyendo una página web y fue una parte importante de mi aprendizaje inicial.

### TransChain

Proyecto desarrollado durante el segundo año de la carrera para gestionar la asignación y el registro de órganos.

La propuesta incorporó tecnologías blockchain mediante la red Hedera con el objetivo de garantizar la trazabilidad y la inmutabilidad de las modificaciones registradas.

El proyecto obtuvo el quinto puesto en una hackathon.

### Sistema de Tracking

Sistema desarrollado para la materia Bases de Datos con el objetivo de gestionar y realizar el seguimiento de envíos entre sucursales.

Es uno de los proyectos más completos de mi formación, ya que integra diseño de bases de datos, reglas de negocio, programación backend, interfaces para diferentes usuarios y administración del ciclo completo de los envíos.

## Evolución

| Proyecto | Etapa académica | Aspecto destacado |
|---|---|---|
| EduData | Primer año | Primera experiencia en desarrollo web |
| TransChain | Segundo año | Aplicación de blockchain con Hedera y quinto puesto en una hackathon |
| Sistema de Tracking | Segundo año | Aplicación integral de bases de datos y desarrollo de software |

## Organización

Cada proyecto dispone de su propia carpeta y documentación. Los archivos se incorporarán progresivamente después de revisar credenciales, datos personales y configuraciones privadas.
