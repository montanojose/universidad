# TransChain

Proyecto universitario desarrollado durante el segundo año de la carrera para gestionar el registro y la asignación de órganos mediante tecnologías blockchain.

## Objetivo

Diseñar una solución que permita registrar las modificaciones realizadas durante el proceso de asignación de órganos, garantizando la trazabilidad, transparencia e inmutabilidad de la información.

## Aplicación de blockchain

El proyecto utiliza la red Hedera para almacenar evidencias verificables de las operaciones realizadas.

La incorporación de blockchain permite:

- Registrar modificaciones de manera trazable.
- Verificar la integridad de la información.
- Evitar alteraciones no detectadas.
- Mantener un historial de operaciones.
- Aumentar la transparencia del proceso.

## Funcionalidades principales

- Registro de información relacionada con órganos.
- Listado y consulta de registros.
- Gestión del proceso de asignación.
- Registro de modificaciones.
- Generación de evidencias verificables mediante Hedera.
- Seguimiento del historial de operaciones.

## Resultado destacado

TransChain obtuvo el quinto puesto en una hackathon, representando una de mis primeras experiencias aplicando tecnologías blockchain en un proyecto funcional.

## Estado

Proyecto académico finalizado.

## Seguridad y privacidad

El repositorio no debe contener información médica real, datos personales, claves privadas, tokens ni credenciales de acceso a Hedera.
