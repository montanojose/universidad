# Sistema de Tracking

Sistema web desarrollado para la materia Bases de Datos durante el segundo año de la carrera.

El proyecto permite gestionar y realizar el seguimiento de envíos entre diferentes sucursales, integrando una base de datos relacional con una aplicación web.

## Objetivo

Desarrollar un sistema que permitiera administrar el ciclo completo de un envío, desde su registro hasta su entrega o retiro, manteniendo un historial de sus movimientos y estados.

## Funcionalidades principales

- Registro de clientes.
- Registro y administración de empleados y choferes.
- Gestión de sucursales.
- Gestión de vehículos y viajes.
- Registro de envíos y paquetes.
- Asociación de envíos con viajes.
- Seguimiento de estados.
- Historial de movimientos.
- Gestión de incidentes.
- Administración de retiros.
- Gestión de devoluciones.
- Paneles adaptados a diferentes tipos de usuarios.

## Base de datos

El proyecto aplica conceptos como:

- Modelo relacional.
- Claves primarias y foráneas.
- Relaciones entre entidades.
- Restricciones de integridad.
- Historial de estados.
- Consultas con múltiples tablas.
- Reglas de negocio.
- Normalización de datos.

## Tecnologías utilizadas

- PHP
- MySQL
- HTML
- CSS
- JavaScript
