# EduData

Sistema web académico desarrollado durante el primer año de la carrera para la materia Programación Web.

## Objetivo

Crear una aplicación que permitiera registrar estudiantes y docentes, además de gestionar la asistencia de los alumnos.

## Funcionalidades principales

- Registro de estudiantes.
- Registro de docentes.
- Consulta de alumnos y profesores.
- Registro de asistencia.
- Almacenamiento de información en una base de datos.
- Formularios para la carga de datos.
- Interfaz web para acceder a las diferentes funcionalidades.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Aprendizajes

EduData fue mi primera experiencia desarrollando una aplicación web conectada a una base de datos. Durante el proyecto trabajé con:

- Estructuración de páginas mediante HTML.
- Creación de estilos con CSS.
- Interacción básica con JavaScript.
- Procesamiento de formularios con PHP.
- Conexión entre PHP y MySQL.
- Operaciones de registro y consulta de información.
- Organización inicial de un proyecto web.

## Estado del proyecto

Proyecto académico finalizado.

Al tratarse de mi primera aplicación web, presenta decisiones técnicas y aspectos de organización que actualmente resolvería de otra manera. Sin embargo, representa una etapa importante de mi aprendizaje y permite observar mi evolución en proyectos posteriores.

## Privacidad

La versión publicada en este repositorio no debe contener información real de estudiantes o docentes, contraseñas ni credenciales de conexión a la base de datos.
